# LangChain Inc.

**LangChain Inc.** là công ty phát triển hệ sinh thái LangChain, được thành lập bởi Harrison Chase. Công ty tập trung vào việc xây dựng công cụ cho các ứng dụng LLM, mã nguồn mở kết hợp dịch vụ thương mại.

## Sản Phẩm

| Sản Phẩm | Loại | Mô Tả |
|----------|------|-------|
| LangChain | Open Source (MIT) | Core framework cho ứng dụng LLM |
| LangGraph | Open Source (MIT) | Agent orchestration framework |
| LangSmith | Commercial | Observability, evaluation, testing platform |
| LangServe | Open Source | Deployment framework |

## Vòng Gọi Vốn

- **Seed Round** (2023) — $10M từ Sequoia Capital và các nhà đầu tư khác
- **Series A** (2023) — $25M, định giá ~$200M
- Tính đến 2025, LangChain Inc. được định giá trên $1B

## Đối Thủ Cạnh Tranh

- **Haystack (deepset)** — Framework NLP mã nguồn mở
- **LlamaIndex** — Framework chuyên về RAG và data indexing
- **Semantic Kernel (Microsoft)** — Framework AI orchestration của Microsoft
- **AutoGen (Microsoft)** — Multi-agent framework
- **CrewAI** — Multi-agent framework đơn giản hơn
